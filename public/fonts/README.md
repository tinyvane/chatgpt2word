# 字体文件目录

这个目录用于存放应用程序使用的字体文件。

## 内置字体

应用程序已包含以下内置字体：

1. **仿宋_GB2312** (仿宋_GB2312.ttf) - 标准公文字体，适用于正文内容
2. **楷体_GB2312** (楷体_GB2312.ttf) - 传统书法风格字体，优雅美观
3. **方正大标宋简体** (方正大标宋简体.ttf) - 适用于标题和重要内容
4. **方正公文小标宋** (方正公文小标宋.TTF) - 适用于公文标题和小标题

## 字体使用说明

这些字体文件已经内置在应用程序中，用户可以：

1. 直接在应用程序中选择使用这些字体
2. 下载并安装这些字体到自己的系统中，以便在其他应用程序中使用

## 字体版权说明

请注意字体的版权和使用许可：

- **仿宋_GB2312** 和 **楷体_GB2312** 是中国国家标准字体，可以在中国大陆地区免费使用
- **方正大标宋简体** 和 **方正公文小标宋** 是方正字库的字体，使用时请遵守相关的版权规定

如果您需要在商业项目中使用这些字体，请确保您拥有合法的使用权限。

## 添加更多字体

如果您需要添加更多字体：

1. 将字体文件(.ttf或.otf格式)放置在此目录中
2. 在应用程序代码中的 `builtinFonts` 数组中添加相应的字体信息
3. 重新构建应用程序

## 字体嵌入说明

当前版本的应用程序支持从fonts目录加载字体，并在浏览器中使用这些字体进行预览。

对于导出的Word文档，应用程序会尝试使用这些字体，但由于技术限制，完全嵌入中文字体可能会导致文档文件过大或不兼容。

为获得最佳效果，建议接收方也安装相同的字体。
