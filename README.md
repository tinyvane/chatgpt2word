# AI文字转Word

一个简单高效的工具，用于将AI生成的无格式文字（如ChatGPT、文心一言、Claude等）一键转换为 Word 文档格式。

## 功能特点

- 简洁直观的用户界面
- 一键将AI生成的无格式文字转换为 Word 文档
- 自定义导出文件名
- 无需手动调整格式，节省时间

## 技术栈

- Vue 3 - 前端框架
- Vite - 构建工具
- docx - Word 文档生成库
- file-saver - 客户端文件保存

## 使用方法

1. 将AI生成的文字内容复制到文本框中
2. 输入自定义的文件名（可选）
3. 点击"导出为 Word"按钮
4. 文档将自动下载到您的设备上

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
